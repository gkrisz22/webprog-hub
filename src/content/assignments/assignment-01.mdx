description: "Készíts egy személyes profil oldalt csak szemantikus HTML használatával. CSS nem szükséges."
---

Készíts egy személyes profil oldalt, amely tartalmazza:

- Oldal cím és metaadatok
- Egy header a neveddel
- Egy navigációs menü (lehetnek üres linkek)
- Egy rólad szóló szekció egy bekezdéssel magadról
- Egy készségek szekció rendezetlen listával
- Egy projektek szekció legalább két projekttel
- Egy footer kapcsolati információkkal

Követelmények:

- Végig szemantikus HTML elemek használata
- Megfelelő címsor hierarchia
- Érvényes HTML5 struktúra
- Nincs inline stílus vagy CSS

Adj le egyetlen HTML fájlt a kurzus platformon a határidőig.
